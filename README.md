# blog

## 技术栈
### K8S集群
1. Ingress Nginx
1. lvs
1. Flannel
1. Redis集群
1. Mysql单机
1. Elasticsearch单机
1. MinIO单机

### 前端相关
1. Vue3
1. TypeScript
1. Element-Plus

### 后端相关
1. Golang
1. Iris 框架
1. Mysql 存储
1. Redis 存储
1. MinIO 文件对象存储
1. Elasticsearch 搜索

### 部署到K8S集群
```shell 
cd scripts && sh k8s_deploy.sh
```

### 传统部署单机
```shell 
cd scripts && sh native_deploy.sh
```
